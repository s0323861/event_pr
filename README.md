## 「告知くん」(Event promotional tool)

イベントの開催＆支援ツールのソースコードです。

デザインはBootstrap3、Font Awesome Iconsを使っています。

Bootstrapのプラグイン(bootstrap-datetimepicker.js、validator.jsなど)も使わせていただきました。


### Usage

  - upload all files to the web server


### Features

  - written in PHP
